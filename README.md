### Immune landscape characterization of premalignant Dysplastic Neurofibromas reveals PD-1 and the Tim-3/Galectin-9 pathway as potential therapeutic targets

This repository contains the code to ensure the traceability of the analysis of single-cell RNA Sequencing data from the raw count matrices to the figures in the associated publication. Please open the [index.html](https://audrey-onfroy.github.io/NF1_patients_analysis/) file in any browser, and navigate through all files in this repository.

The results behind this analysis are *under revision*:

> Brunet *et al.*, Immune landscape characterization of premalignant Dysplastic Neurofibromas reveals PD-1 and the Tim-3/Galectin-9 pathway as potential therapeutic targets, doi: not yet available
